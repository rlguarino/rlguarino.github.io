Projects

μKernel

A Micro Kernel operating system that provides a simple REST/Http interface.

The goal of this operating system was to build a kernel API that other developers could use to add functionality to the system without creating new system calls. What started as an attempt to design a VFS turned into a kernel module system that emulated a REST service. The Kernel provided a set of six system calls for registering and interacting with resources provided by kernel modules.

Keylogger

A windows KeyLogger built as an experiment in design and implementation of malicious software. The KeyLogger was built using redundant components for resilience with a goal to remain undetected. The KeyLogger preformed great in trials and was never disabled or detected by the targets.

Intelligent Scan Detection

An Intrusion Detection program that is designed to detect and respond to port scans. Using a Neural Network to detect an ongoing port scan the software takes action to notify user and tale protective action in real time.
